# 设计阶段指南

## 家庭背景

- **常住人口**：两口之家，均不到 30 岁，无孩
- **偶住**：父母偶尔来小住几天（一年 2-3 次，每次 3-5 天）
- **设计定位**：年轻现代、灵活可变、控制预算

---

## 一、需求梳理（开工前达成一致）

- [ ] 主卧：舒适大床 + 充足衣物收纳 + 主卫使用便利
- [ ] 第三室：书房为主 + 偶尔客房功能（沙发床/墨菲床）
- [ ] 客厅：观影+社交为主，考虑投影或大屏电视
- [ ] 是否需要在家办公 → 决定书房桌位长度（单人 1.2m / 双人 2m+）
- [ ] 烹饪习惯 → 开放式/半开放厨房？（年轻人倾向开放，但需考虑燃气开通要求）
- [ ] 收纳重点：两人换季衣物、行李箱、清洁工具、囤货
- [ ] 智能家居：智能灯、电动窗帘、智能门锁、扫拖机器人（年轻人偏好）
- [ ] 空调方案：风管机+挂机，还是全挂机（预算决定）

---

## 二、平面布局规划（116㎡ 三室两卫 · 年轻两口版）

### 主卧
- 1.8m 床 + 双床头柜 + 满墙衣柜（深度 60cm）
- 主卫门口不要正对床（如果户型有这个问题，用衣柜或隔断遮挡）

### 次卧（预留客房/未来儿童房）
- 先不买床，空置或放健身器材/收纳架
- 或者放一张 1.5m 床 + 小衣柜（父母来时用）
- 后期可灵活升级为儿童房

### 第三室：书房 + 客卧
- **推荐方案**：双人书桌（2m+） + 书柜 + 沙发床（拉开 1.5m）
- 沙发床平时是沙发，父母来了拉开睡
- 如果空间宽裕，可考虑墨菲床（壁床），但预算较高

### 客厅
- 沙发 + 电视/投影 + 满墙收纳柜
- 年轻两口更适合：整墙柜体（收纳+展示+电视嵌入）
- 阳台如果打通纳入客厅，显大且采光好

### 餐厅
- 日常 2-3 人吃饭，1.2-1.4m 餐桌足够
- 父母来时 4 人，1.2m 桌也能坐
- 餐边柜必做：水吧台（咖啡机、水杯、零食）

### 厨房
- L 型或 U 型，视户型定
- 洗碗机必装（年轻人刚需）
- 台面插座留够（咖啡机、电饭煲、空气炸锅、微波炉）

### 卫生间
- 公卫干湿分离（洗手台外移更实用）
- 主卫：淋浴房即可，不需要浴缸（占空间、使用率低）

---

## 三、风格定位（年轻两口推荐）

### 首推：现代简约 + 奶油元素
- 主调：奶白 + 浅木色 + 浅灰
- 地面：浅灰柔光砖（750×1500mm 全屋通铺显大）
- 墙面：全屋奶白/杏灰色乳胶漆
- 顶面：双眼皮吊顶（比全屋平顶省 30-50%）
- 特点：干净耐看、预算友好、不挑家具

### 备选：原木风
- 地面：木纹砖或强化地板（比实木便宜且好打理）
- 家具：白橡木/榉木色，藤编元素
- 更适合有小孩家庭，两人住也可以

### 不推荐
- 新中式：风格偏稳重，不适合 20+ 年轻人
- 法式奶油：线条和弧形造型多，全包预算兜不住
- 工业风：郑州干燥+灰尘大，裸露墙面不好打理

## 四、收纳规划（两人量级）

| 空间 | 收纳形式 | 说明 |
|------|----------|------|
| 玄关 | 鞋柜（深度 35cm） | 两人约 40-50 双鞋 + 客拖 |
| 客厅 | 满墙电视柜（深度 35cm） | 杂物、囤货、书籍、展示 |
| 餐厅 | 餐边柜（深度 35-40cm） | 小家电、零食、水吧 |
| 厨房 | 地柜 + 吊柜 | 两人做饭够用，不用过度配置 |
| 主卧 | 满墙衣柜（深度 60cm） | 两人衣物+床品，挂区为主 |
| 次卧 | 暂不配衣柜 | 后期需要再买成品柜 |
| 书房 | 书柜/展示柜 | 书籍+文件+手办 |
| 阳台 | 阳台柜 | 清洁工具、行李箱、囤货 |
| 卫生间 | 镜柜 + 浴室柜 | 洗漱护肤+清洁剂 |

## 五、灯光设计

### 预算友好方案：主灯 + 局部射灯
| 区域 | 方案 | 色温 |
|------|------|------|
| 客厅 | 简约主灯 + 沙发背景墙射灯×3 | 3500K |
| 餐厅 | 吊灯（距桌面 75cm） | 3000K |
| 卧室 | 简约吸顶灯 + 床头吊灯 | 3000K |
| 书房 | 吸顶灯 + 书桌台灯 | 4000K |
| 厨房 | 平板灯 + 吊柜下手扫感应灯带 | 4000K |
| 卫生间 | 防水筒灯 + 镜前灯 | 4000K |
| 过道 | 筒灯 | 3500K |

> 全屋无主灯（满吊平顶+磁吸轨道）成本高 3000-5000，预算紧张建议用主灯方案。

## 六、开关插座规划（两人场景）

### 容易漏掉的关键插座
- [ ] 玄关：烘鞋器 + 智能门锁充电口
- [ ] 窗帘盒：电动窗帘插座（客厅+主卧，共 2 个）
- [ ] 阳台：洗衣机 + 烘干机（独立 2 个插座）
- [ ] 厨房水槽下：洗碗机 + 净水器 + 垃圾处理器（可选）
- [ ] 卫生间：智能马桶盖（2 个）+ 电热毛巾架（1 个）
- [ ] 弱电箱内：光猫 + 路由器电源（2 个）
- [ ] 书桌区：桌面以上留 3-4 个插座（电脑+显示器+充电）
- [ ] 家政柜/阳台柜：吸尘器/洗地机充电
- [ ] 客厅沙发两侧：手机充电（带 USB 口的插座）

## 七、全包模式下的设计注意事项

因为是熟人工作室全包，设计方面：
- 工作室一般有固定合作的"设计师"（实际可能是工长画图），设计水平不一定高
- 建议自己用 **酷家乐** 先做一版布局，明确各空间怎么摆
- 把开关插座点位、灯位图自己画一份给到工长，不要全靠他们定
- 小红书搜同户型案例，把喜欢的图存下来直接给工长看（比口头描述有效 10 倍）
- 柜体内部格局自己画简图（挂衣区/叠放区比例），避免工长按"惯例"做
