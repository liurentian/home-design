# 设计阶段指南

## 家庭背景

- **常住人口**：两口之家，均不到 30 岁，无孩
- **偶住**：父母偶尔来小住几天（一年 2-3 次，每次 3-5 天）
- **设计定位**：年轻现代、灵活可变、控制预算

## 一、需求梳理（开工前达成一致）

- [ ] **入户花园**：独立玄关 + 储物（鞋柜+换季物品+行李箱），不做端景墙
- [ ] **阳台**：客厅与南次卧之间保留贯通，不砌墙隔开
- [ ] **主卧**：L 形衣柜（利用进深+开间转角），1.8m 床靠墙角，不做床头柜
- [ ] **北次卧**：书房为主 + 偶尔客房功能（沙发床），与主卫相邻墙做隔音
- [ ] **南次卧**：1.5m 床 + 小衣柜，父母偶住，未来灵活调整
- [ ] **客厅**：观影+社交为主，满墙电视柜收纳
- [ ] **是否需要在家办公** → 决定北次卧书桌长度（单人 1.2m / 双人 2m+）
- [ ] **烹饪习惯** → 开放式/半开放厨房？（年轻人倾向开放，但需考虑燃气开通要求）
- [ ] **智能家居**：智能灯、电动窗帘、智能门锁、扫拖机器人（年轻人偏好）
- [ ] **空调方案**：全挂机（预算最优，4 台挂机 + 1 台柜机）

## 二、平面布局规划

### 入户花园（4000×1700mm · 西北角）

入户门朝西内开，进门正对东墙落地窗。南侧为通往客餐厅的过道。

**布局方案**：沿西墙 + 北墙做 L 形柜体
- **西墙（门旁）**：鞋柜，深度 35cm，容纳 40-50 双鞋 + 客拖。底部留空 15cm 放常穿鞋，中间镂空放钥匙/口罩
- **北墙**：储物柜，深度 60cm，收纳换季鞋、行李箱、囤货、清洁工具
- **东墙落地窗**：保留，不遮挡。可在窗前放换鞋凳
- **过道（南侧）**：保持通畅，不设柜体

> 入户花园是室内到室外的缓冲，地面用瓷砖（耐磨好打理），与客餐厅地板接口做压条过渡。

### 客餐厅（3400×~9000mm · 西侧竖厅）

南北通透竖厅，北连厨房和入户过道，南连阳台。

**餐厅区（北段）**：
- 餐桌 1.2-1.4m，靠西墙或居中，日常 2-3 人，父母来 4 人也能坐
- 餐边柜靠东墙（公卫外墙侧），深度 35-40cm，水吧台（咖啡机、水杯、零食）

**客厅区（南段）**：
- 沙发靠东墙（与主卧隔墙），电视嵌入西墙满墙收纳柜
- 满墙电视柜（西墙）：深度 35cm，收纳杂物/囤货/书籍/展示
- 茶几选轻便可移动款，留出活动空间

### 厨房（2300×2000mm · L 型）

门朝东对餐厅。L 型操作台，冰箱放厨房内或餐厅角落。

- 灶台靠北墙，水槽靠西墙，形成洗-切-炒动线
- 洗碗机装水槽下方（年轻人刚需）
- 吊柜做到顶，增加收纳
- 门用透明推拉门：视觉开放 + 满足燃气开通要求
- 台面插座留够（咖啡机、电饭煲、空气炸锅、微波炉）

### 公卫（1500×2100mm · 干湿分离）

洗手台外移至走廊，原空间做马桶 + 淋浴区。
- 洗手台宽度约 800-900mm，镜柜收纳
- 淋浴区做挡水条 + 浴帘或玻璃隔断

### 主卧 + 主卫（3000×5200mm · 东南）

**核心问题**：开间仅 3000mm。常规布局「满墙衣柜 600 + 1.8m 床 ~1850 + 双床头柜 2×400 = 3250」放不下。

**L 形衣柜方案**：
```
      北 → (主卫)
    ┌──────────────┐
    │ 主卫         │
    │ 1500×3100    │
    ├────┬─────────┤
    │    │ L形衣柜 │  ← 短边沿北墙 1500/2000
    │    │         │
    │ 床 │         │  ← 长边沿东墙，利用 5200 进深
    │1.8m│         │
    │    │         │
    │    ├────┬────┤
    │    │ 门 │    │  ← 门在南墙西侧
    └────┴────┴────┘
    ← 3000 →
```

- L 形衣柜：短边沿北墙（主卫外墙下方），长边沿东墙。既满足收纳量，又不挤压 3000 开间
- 1.8m 床靠西南角（床头朝东或朝北），不做床头柜
- 床边留 ≥600mm 过道
- 主卫门不要正对床，L 形衣柜恰好形成遮挡

### 南次卧（2800×3150mm · 中南）

南向采光好，连接阳台。当前用途：父母偶住。

- 1.5m 床 + 小型成品衣柜（或暂不配，后期加）
- 通阳台的法兰门保留，配窗帘
- 电动窗帘插座预留（阳台侧）
- 阳台连通客厅，南次卧窗外为阳台休闲区，注意隐私（可装纱帘或百叶帘）

### 北次卧（2700×3200mm · 东北）

书房为主 + 偶尔客房。紧邻主卫，**西墙与主卫共用**。

- 双人书桌 2m+ 靠东墙或北墙
- 书柜/展示柜靠墙
- 沙发床（拉开 1.5m），父母来时可睡
- **隔音处理**：与主卫相邻的西墙加隔音棉 + 双层石膏板，减少管道冲水噪音

### 阳台（6000×1600mm · 贯通）

横跨客厅和南次卧，不砌隔墙，保留贯通。

**客厅端（西段）**：
- 洗衣机 + 烘干机叠放或并排
- 家政柜：吸尘器/洗地机充电、清洁剂收纳
- 电动晾衣架

**南次卧端（东段）**：
- 休闲区：休闲椅 + 小边几 + 绿植
- 南次卧可通过法兰门直接进入阳台

**封闭方案**：断桥铝落地窗，客厅端和南次卧端各留一扇可开启窗，保证通风。

## 三、风格定位（年轻两口推荐）

### 首推：现代简约 + 奶油元素
- 主调：奶白 + 浅木色 + 浅灰
- 地面：浅灰柔光砖（750×1500mm 全屋通铺，含入户花园和阳台）
- 墙面：全屋奶白/杏灰色乳胶漆
- 顶面：双眼皮吊顶（比全屋平顶省 30-50%）
- 特点：干净耐看、预算友好、不挑家具

### 备选：原木风
- 地面：木纹砖或强化地板（比实木便宜且好打理）
- 家具：白橡木/榉木色，藤编元素
- 更适合有小孩家庭，两人住也可以

### 不推荐
- 新中式：风格偏稳重，不适合 20+ 年轻人
- 法式奶油：线条和弧形造型多，全包预算兜不住
- 工业风：郑州干燥+灰尘大，裸露墙面不好打理

## 六、开关插座规划

### 容易漏掉的关键插座
- [ ] **入户花园**：烘鞋器 + 智能门锁充电 + 柜内吸尘器充电
- [ ] **窗帘盒**：电动窗帘插座 ×3（客厅 + 主卧 + 南次卧）
- [ ] **阳台**：洗衣机 + 烘干机（独立 2 个）+ 电动晾衣架 + 扫拖机器人基座
- [ ] **厨房水槽下**：洗碗机 + 净水器
- [ ] **卫生间**：智能马桶盖 ×2 + 电热毛巾架（主卫 1 个）
- [ ] **弱电箱内**：光猫 + 路由器电源（2 个）
- [ ] **北次卧书桌区**：桌面以上 3-4 个（电脑+显示器+充电），避开与主卫相邻的西墙
- [ ] **家政柜/阳台柜**：吸尘器/洗地机充电
- [ ] **客厅沙发两侧**：手机充电（带 USB 口的插座）
- [ ] **餐边柜**：咖啡机、热水壶等小家电（3-4 个）

## 七、全包模式下的设计注意事项

因为是熟人工作室全包，设计方面：
- 工作室一般有固定合作的"设计师"（实际可能是工长画图），设计水平不一定高
- 建议自己用 **酷家乐** 先做一版布局，明确各空间怎么摆
- 把开关插座点位、灯位图自己画一份给到工长，不要全靠他们定
- 小红书搜同户型案例，把喜欢的图存下来直接给工长看（比口头描述有效 10 倍）
- 柜体内部格局自己画简图（挂衣区/叠放区比例），避免工长按"惯例"做

### 需要明确写进合同的特殊项
- [ ] **北次卧隔音墙**：隔音棉 + 双层石膏板，确认是否在全包范围内
- [ ] **主卧 L 形衣柜**：提供尺寸草图，确认柜体深度/格局/板材
- [ ] **阳台封闭**：确认物业是否允许，断桥铝品牌和规格
- [ ] **阳台柜**：家政柜是否在全包范围内
