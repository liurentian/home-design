# 全屋智能 + 网络方案

## 一、目标与范围

**目标**：装修阶段做好预留，入住后用最低预算实现全屋语音控制（灯光/空调/窗帘/家电）。

**不做**：全屋调光、背景音乐、安防传感器（成本高、使用率低）。

**预算**：¥1,500-2,000（不含已持有的 HomePod、智能风扇、台灯）。

## 三、网络基础

### 为什么必须 Mesh

户型南北纵深约 12.7m。弱电箱在入户花园西北角，主卧在东南角，中间隔 3+ 面墙。单路由 5G WiFi 到主卧基本没信号。

### Mesh 有线回程方案

```
弱电箱(入户花园)
  │
  ├─ 网线×2 → 客厅电视柜 → 主路由（小米 AX3000T）
  │              ├─ 有线覆盖：电视/游戏机/小爱Pro
  │              └─ 无线覆盖：客厅/餐厅/厨房/阳台
  │
  └─ 网线×1 → 北次卧书桌 → 子路由（小米 AX3000T）
                 ├─ 有线给台式机/NAS
                 └─ 无线覆盖：北次卧/主卧/南次卧/主卫
```

子路由放北次卧比放主卧好——位置居中，向南穿一面墙到主卧+南次卧，信号优于客厅穿三面墙。

## 四、生态方案：米家设备 + Siri 双控

### 为什么用桥接而不是纯米家

- HomePod 已经买了，纯米家的话 HomePod 吃灰
- 米家设备便宜（插座 ¥49、开关 ¥69），HomeKit 原生设备贵 3-5 倍
- HomeBridge 免费软件，搭一次就再也不用管

### HomeBridge 怎么搭

| 方案 | 成本 | 上手难度 |
|------|------|----------|
| 家里有 NAS（群晖/威联通） | ¥0，Docker 跑容器 | 低（15 分钟配好） |
| 树莓派 3B/4B | ~¥200 | 中（刷系统 + 安装） |
| 旧安卓手机（Android 5.0+） | ¥0，装 Termux | 低中（手机常年插电） |
| 长期开机的 Windows 台式机 | ¥0 | 低 |

有 NAS 最优，没有就旧手机弄一下。

### 控制链路

```
"小爱同学，关客厅灯"
  → 小爱音箱 Pro
  → 米家服务器
  → Zigbee → 智能开关 → 灯关了

"Hey Siri，关客厅灯"
  → HomePod / iPhone
  → HomeKit
  → HomeBridge（翻译指令）
  → 米家服务器
  → Zigbee → 智能开关 → 灯关了
```

两种语音都能控所有设备，互不冲突。

## 八、最终效果

### 日常场景

| 场景 | 触发方式 | 效果 |
|------|----------|------|
| 回家 | 开智能门锁 / "Hey Siri 我回家了" | 入户灯亮、客厅灯亮、窗帘关闭（夜间） |
| 离家 | "小爱同学，离家模式" / 手机点一下 | 全屋灯关、空调关、窗帘开、扫地机器人出动 |
| 睡觉 | "Hey Siri 晚安" | 全屋灯关、主卧空调 26°、客厅窗帘关闭 |
| 观影 | "小爱同学，电影模式" | 客厅灯关/暗、窗帘关、电视开 |
| 调空调 | "小爱同学，主卧空调 26 度" 或 "Hey Siri 调空调" | 对应房间空调响应 |
| 开关窗帘 | "小爱同学，开客厅窗帘" | 对应窗帘自动开/关 |

### 自动化（无需语音）

- 日出后自动开客厅窗帘（配合定时）
- 室温 >28° 自动开卧室空调
- 离家后扫地机器人自动打扫
- 智能门锁开锁 → 入户灯亮（配合人体传感器）

## 九、不做的事

| 不做 | 原因 |
|------|------|
| 全屋调光调色灯 | 成本 +¥2000-3000，且需要支持 Zigbee 的灯，选择少 |
| 背景音乐系统 | 吸顶喇叭+功放 ¥3000+，HomePod+小爱够用 |
| 门磁/窗磁/人体传感全覆盖 | 两口之家没必要，真有需求后期 49 元加一个 |
| 指纹/人脸门锁接入自动化 | 涉及安全，开门联动需小心调试，先不折腾 |
| 中央空调接入智能 | 格力风管机用红外控即可，不动原装线控 |
