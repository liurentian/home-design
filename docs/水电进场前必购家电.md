# 水电进场前必须敲定的家电清单

> **原则：凡是影响水电定位、开槽布线的家电，水电进场前必须确定型号。先定型号再定水电，不然后期改不回来。**

---

## 为什么水电前就要选好？

水电属于隐蔽工程，一旦封槽贴砖，改一个插座位置就要砸墙。这些家电决定了：

- 插座位置 + 数量 + 高度
- 进出水口位置
- 排水口位置
- 是否需要独立回路 / 4 平方线 / 16A 插座
- 燃气管路走向
- 排烟 / 排气孔位置和孔径
- 预埋管道（如电视穿线管、管线机 PE 管）

---

## 分类清单

### 标记说明

| 标记 | 含义 |
|------|------|
| 🔴 最高 | 不定型号就没法拉水电，必须提前定 |
| 🟡 高 | 不提前定也行但后期会很麻烦，强烈建议提前定 |
| 🟢 中 | 可以预留标准点位，后期再选型号影响不大 |

---

## 一、空调 🔴 最高

**为什么水电前必须定：** 空调类型决定了外机位置、铜管/冷凝水管走向、内机电源线规格、打孔位置，全都不一样。

### 三种方案对比

| | 中央空调 | 风管机+挂机 | 全挂机（推荐） |
|------|------|------|------|
| 价格 | ¥25000-35000 | ¥16000-22000 | ¥11000-14900 |
| 吊顶 | 全屋吊顶 | 局部吊顶 | 不吊顶 |
| 层高影响 | 较大 | 局部 | 无 |
| 美观 | 最好 | 中等 | 一般（管子外露） |
| 能耗 | 较高（全开才省） | 中等 | 最低（单开单控） |
| 后期维保 | 麻烦 | 部分麻烦 | 方便 |
| 适用场景 | 预算充足+常全开 | 折中方案 | 20 万预算首选 |

**给本项目的推荐：全挂机方案**

- 客厅：3 匹一级变频柜机 → 需预留 **16A 插座**（距地 30cm）+ 空调孔（距地 15-20cm）
- 卧室 ×3：1.5 匹一级变频挂机 → 需预留 **16A 插座**（距顶 30cm）+ 空调孔
- 外机位：确认开发商预留机位数够不够 4 台外机

**水电要做的：**
- 每个空调走 **单独回路**（4 平方线）
- 挂机插座预留在靠近挂机位置（避开床头正上方）
- 柜机插座预留在内机附近低处
- 空调孔在油工前打好（水电阶段也可以先打），注意向外倾斜防雨水倒灌

**预算：¥11000-14900**

---

## 二、燃气热水器 🔴 最高

**为什么水电前必须定：** 决定了燃气管走向、冷热水管间距、排烟孔位置（承重墙不能随便打，要提前报物业）。

### 选型要点

| 参数 | 建议 |
|------|------|
| 类型 | 强排式（主流，装在厨房或生活阳台） |
| 升数 | 16L（一厨两卫，能同时洗澡+洗碗） |
| 恒温 | 水量伺服器（不是燃气比例阀） |
| 品牌 | 万和/海尔/美的/万家乐 |
| 预算 | ¥1500-2500 |

> 为什么不选林内/能率（¥4000-6000）？国产 16L 水量伺服款的技术已经成熟，多花一倍钱提升不大，20 万预算优先省着花。

### 水电预留

| 项目 | 要求 |
|------|------|
| 冷热水管 | 左热右冷，中心间距 15cm |
| 燃气管 | 提前埋好，黄管或不锈钢波纹管，水电阶段预埋 |
| 排烟孔 | 孔径 60-80mm（不同型号不同，定了型号就知道），距吊顶 ≥10cm |
| 插座 | 1 个 10A 三孔（单独回路），距地 130-150cm（在热水器侧面或下方，不被水管挡住） |
| 位置 | 一般装在厨房靠窗位置或生活阳台，尽量靠近用热水点（两个卫生间） |

**预算：¥1500-2500**

---

## 三、烟机灶具 🔴 最高

**为什么水电前必须定：** 决定了烟道开孔位置、烟机插座位置。不同烟机（侧吸/顶吸/欧式）插座高度要求不同，装错了就偏。

### 选型要点

| 参数 | 建议 |
|------|------|
| 类型 | 侧吸（不碰头，吸烟口离锅近） |
| 风量 | ≥20m³/min（爆炒型家庭） |
| 风压 | ≥400Pa（高层中间楼层尤其重要，防倒灌） |
| 灶具火力 | ≥5.0kW（中式爆炒） |
| 品牌 | 华帝/美的/万和 |
| 预算 | ¥3000-4500（套装） |

### 水电预留

| 项目 | 要求 |
|------|------|
| 烟机插座 | 10A 三孔，预留在吊顶内或烟机罩内（不要露在外面），距地 200-220cm |
| 灶具 | 灶下一般不需要插座（除非买的带点火器的特殊款），但灶下橱柜留一个备用 |
| 烟道止逆阀 | 让瓦工贴砖时就装好，密封打胶 |
| 排烟管 | 提前预埋到吊顶内，接止逆阀 |
| 灶具侧气源 | 燃气管到位 |

> **注意**：如果烟机插座留低了，露在外面很难看，高了藏在吊顶里最干净。水电交底时把烟机型号告诉电工，电工知道留多高。

**预算：¥3000-4500**

---

## 四、洗碗机 🟡 高

**为什么水电前最好定：** 需要进水、排水、插座都在隔壁柜体（通常放水槽隔壁），下水要接三通。

### 选型要点

| 参数 | 建议 |
|------|------|
| 类型 | 嵌入式 10-13 套（两人 10 套够，想洗锅就 13 套） |
| 烘干 | 热风烘干（比冷凝干得透） |
| 品牌 | 美的/华凌/海尔 |
| 预算 | ¥2500-3500 |

### 水电预留

| 项目 | 要求 |
|------|------|
| 位置 | 紧挨水槽柜，方便接进水排水 |
| 插座 | 10A 三孔，留在隔壁柜体（不要留洗碗机正后方，不然机器推不进去），距地 30-40cm |
| 进水 | 水槽角阀处加三通，分出洗碗机进水 |
| 排水 | 水槽下水加三通，接洗碗机排水管 |
| 开孔 | 橱柜侧板开 50mm 孔穿管线 |

> **即使现在不买也先把水电留好**——留了不装就是多一个插座+堵头，不留以后想装得拆橱柜。

**预算：¥2500-3500**

## 五、冰箱 🟡 高

**为什么水电前最好定：** 决定了插座位置（必须单独回路，出差关总闸时冰箱不断电），以及冰箱位宽度是否够。

### 选型要点

| 参数 | 建议 |
|------|------|
| 容量 | 450-500L |
| 门型 | 十字对开或法式四门 |
| 制冷 | 风冷无霜 |
| 品牌 | 美的/容声/海尔 |
| 预算 | ¥3000-4500 |

### 水电预留

| 项目 | 要求 |
|------|------|
| 插座 | 10A 三孔，**单独回路**（出差拉闸不关冰箱），距地 130-150cm（高于冰箱高度）或留在侧面柜体 |
| 位置 | 厨房内或餐厅旁，注意开门方向和空间 |
| 散热 | 两侧各留 5-10cm 散热空间 |

**预算：¥3000-4500**

## 六、洗衣机+烘干机 🟡 高

**为什么水电前最好定：** 决定了进水龙头高度、排水方式、插座数量和位置。洗烘叠放和并排走的水电完全不一样。

### 选型要点

| | 洗衣机 | 烘干机 |
|------|------|------|
| 容量 | 10kg | 10kg |
| 类型 | 滚筒 | 热泵式 |
| 品牌 | 小天鹅/海尔 | 小天鹅/海尔 |
| 预算 | ¥1800-2500 | ¥2500-3500 |

> 不推荐洗烘一体机：烘得慢（3 小时+）、容量缩水（10kg 变 6kg）、毛絮堆积难清理。

### 水电预留

| 项目 | 要求 |
|------|------|
| 摆放方式 | 并排（空间够）或叠放（空间紧）→ 决定了插座和水龙头位置 |
| 进水 | 1 个洗衣机水龙头（烘干机不需要进水） |
| 排水 | 洗衣机地漏（专用，带存水弯防臭） |
| 插座 | 并排 2 个 10A 三孔；叠放预留 2 个在侧面（高度约 120-130cm，方便插拔） |
| 位置 | 生活阳台或卫生间干区 |

**预算：¥4300-6000**

---

## 七、智能马桶 / 马桶盖 🟡 高

**为什么水电前最好定：** 决定了马桶旁有没有插座。没有的话后期只能走明线，丑且不安全。

### 选型要点

| | 公卫 | 主卫 |
|------|------|------|
| 方案 | 普通马桶 + 智能马桶盖 | 轻智能马桶或普通+智能盖 |
| 品牌 | 九牧/恒洁/箭牌 | 同左 |
| 预算 | ¥1500-2500 | ¥1500-2500 |

> 轻智能马桶 = 座圈加热 + 离座冲水，没有冲洗功能。智能马桶盖 = 座圈加热 + 冲洗 + 暖风，功能更全。两者都要插座。

### 水电预留

| 项目 | 要求 |
|------|------|
| 插座 | 10A 三孔带防溅盒，距地 40-50cm，在排污管中心线左/右 30-40cm 处 |
| 角阀 | 马桶旁，一进一出 |
| 坑距 | 305mm 还是 400mm → 交房时量好，告诉买马桶的人 |

**预算：¥3000-5000（两个马桶）**

## 八、RO 反渗透净水器 🟢 中

**为什么建议水电前定：** 水槽下方需要预留插座。如果后期再加管线机，还要预埋 PE 管。

### 选型要点

| 参数 | 建议 |
|------|------|
| 通量 | 600G-800G（接水不用等） |
| 品牌 | 小米/美的 |
| 预算 | ¥1200-2000 |

### 水电预留

| 项目 | 要求 |
|------|------|
| 水槽下插座 | 2-3 个 10A 五孔（净水器 + 垃圾处理器 + 备用），带防溅盒 |
| PE 管（如装管线机） | 从水槽下到管线机位置（如餐边柜），预埋 4 分管或 2 分 PE 管套管 |

**预算：¥1200-2000**

## 九、扫拖机器人（自动上下水版）🟢 中

**为什么建议水电前定：** 自动上下水版需要进水口和排水口，得在水电阶段预留。只买水箱版不需要。

### 选型要点

| 参数 | 建议 |
|------|------|
| 品牌 | 石头 P 系列/追觅 S 系列 |
| 预算 | ¥2500-3500 |

### 水电预留

| 项目 | 要求 |
|------|------|
| 自动上下水版 | 基站位置下方留进水角阀 + 排水管（地漏或墙排） + 10A 插座 |
| 位置 | 一般放客厅阳台或生活阳台，靠近水源和地漏 |
| 水箱版 | 只需要一个 10A 插座 |

> 即使先不买，建议也留一个插座在阳台/柜子底部。

**预算：¥2500-3500**

## 十、浴霸 🟢 中

**为什么建议水电前定：** 决定了吊顶内预留几根线（浴霸款 vs 普通款不一样），以及开关是普通还是智能面板。

### 选型

| 参数 | 建议 |
|------|------|
| 类型 | 风暖（不要灯暖，刺眼且取暖范围小） |
| 品牌 | 欧普/美的/雷士 |
| 预算 | ¥400-800/台 |

### 水电预留

| 项目 | 要求 |
|------|------|
| 吊顶预留线 | 传统浴霸需 5-6 根控制线（取暖×2+照明+换气+吹风）+ 零线；智能浴霸一般 3 根（火零地） |
| 开关底盒 | 预留 86 底盒（普通双开）或智能面板位 |
| 排气管 | 吊顶内预埋排气管到室外排气孔，孔径 100mm |

**预算：¥800-1600（两台）**

## 十一、其他需要水电预留的家电 🟢 中

### 电视壁挂

| 预留 | 做法 |
|------|------|
| 预埋 50 管 | 从电视位到电视柜/插座位，穿 HDMI/电源线/网线，外面看不到线 |
| 插座 | 电视位 3-4 个五孔 + 1 个网口 |
| 高度 | 中心距地 100-110cm（坐沙发平视） |

### 电动窗帘

| 预留 | 做法 |
|------|------|
| 插座 | 窗帘盒内预留 10A 五孔，距顶 10-15cm |
| 窗帘盒宽度 | 单轨 15cm，双轨（帘+纱）25cm |
| 位置 | 客厅/主卧建议留，其余可后加充电款 |

### 全屋网络（Mesh 组网）

| 预留 | 做法 |
|------|------|
| 弱电箱 | 换大号（开发商自带的小），预留 2 个插座给光猫+交换机 |
| 网线 | **每个房间拉一根超六类网线**（书房拉两根），客厅电视位一根，吊顶预留一根给吸顶 AP |
| 到弱电箱 | 所有网线汇集到弱电箱 |

### 智能开关（零火线）

| 预留 | 做法 |
|------|------|
| 底盒 | 所有开关底盒 **预留零线**（传统开关只走火线，智能开关需要零火供电） |
| 底盒深度 | 用 5cm 加深底盒，智能开关模块大，浅底盒塞不进去 |

### 垃圾处理器

| 预留 | 做法 |
|------|------|
| 插座 | 水槽下多留 1 个 10A 五孔 |

## 时间节点建议

```
收房后 → 跑建材市场/线上比价
         ↓（1-2 周内敲定型号）
确定空调+热水器+烟灶+洗碗机型号
         ↓
水电交底（拿着型号和安装尺寸图给师傅）
         ↓
水电开槽布管布线
         ↓
水电验收（对着 checklist 逐项核对）
```

> **千万不要：水电都进场了还没定型号。** 师傅问你"空调插座留哪儿"，你说"还没想好"，师傅只能按通用做法留——通用做法往往不适用你的户型，后面改起来代价很大。
