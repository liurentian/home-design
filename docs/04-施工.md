# 施工全流程指南（全包模式 · 熟人工作室）

## 施工模式

**全包给郑州本地小工作室（家里熟人承接）**。

全包通常包含：设计 + 辅材 + 人工 + 主材（瓷砖/地板/门/卫浴/橱柜/集成吊顶/开关插座/窗台石等）
需确认是否包含：全屋定制柜体、美缝、灯具、窗帘

---

## 零、开工前最重要的事：合同与范围

即使是熟人，也必须白纸黑字写清楚。口头承诺是后期扯皮的根源。

### 必须写进合同/清单的项

- [ ] **工程范围**：列出所有包含的项目（拆改、水电、防水、贴砖、吊顶、油工、安装）
- [ ] **主材清单**：瓷砖品牌/型号/尺寸、地板品牌/型号、卫浴品牌/型号、门品牌/材质、橱柜板材/台面材质、开关插座品牌系列
- [ ] **不包含项目**：明确哪些不包（家电、家具、窗帘、灯具、美缝等），避免"我以为包的"
- [ ] **水电点位**：约定包含多少个插座/开关/网口/下水，超出怎么算
- [ ] **工期**：起止日期，逾期怎么处理
- [ ] **付款节点**：分几期、每期比例、每个节点的验收标准
- [ ] **质保**：水电隐蔽工程质保几年、表面工程质保几年
- [ ] **增项规则**：合同外新增项目怎么计价

### 熟人装修特别注意

- 不要因为"是熟人"就不好意思提要求，现在不好意思，住进去每天别扭
- 价格可以给亲情价，但标准不能降
- 验收标准一视同仁，该返工的返工
- 付款照样按节点走，不要一次性付清
- 如果长辈参与沟通，让你爱人负责对接，减少摩擦

---

## 施工总流程

```
拆改 → 水电 → 地暖打压 → 防水 → 瓦工 → 木工 → 油工 → 安装 → 保洁 → 软装进场
```

---

## 一、第一阶段：拆改与砌墙（3-5 天）

重点盯：
- [ ] 确认哪面墙能动（**承重墙、剪力墙绝对不能动**）
- [ ] 拆墙前物业审批通过
- [ ] 新砌墙与老墙连接处 **必须植筋**（每隔 50cm 高度植一根）
- [ ] 新旧墙体交接面 **必须挂钢丝网** 再抹灰
- [ ] 门洞上方 **必须加过梁**
- [ ] 如果是非耐水腻子，必须铲掉重做（泼水手搓，掉白浆则必须铲）

---

## 二、第二阶段：水电改造（5-7 天）

**全包模式下，水电是最容易被"减配"的环节**。材料品牌和回路数量要提前确认。

### 确认电线水管品牌
- [ ] 电线品牌：郑州本地常用——郑州三厂、金水、上上等（必须是国标线）
- [ ] 水管品牌：伟星、日丰、金牛等（伟星有上门打压质保服务）

### 回路规划（要求施工方照此做）
| 回路 | 电线规格 | 说明 |
|------|---------|------|
| 照明 | 1.5mm² | 全屋一路 |
| 普通插座 | 2.5mm² | 客餐厅+卧室，分 2-3 路 |
| 厨房插座 | 4mm² | 独立回路 |
| 冰箱 | 2.5mm² | **独立回路**（出远门只留冰箱） |
| 客厅空调 | 4mm² | 独立回路 |
| 主卧空调 | 4mm² | 独立回路 |
| 北次卧空调 | 4mm² | 独立回路 |
| 主卫 | 4mm² | 独立回路 |
| 公卫 | 4mm² | 独立回路 |

### 施工巡查要点
- [ ] 强弱电交叉处包锡纸
- [ ] 线管进底盒用锁扣
- [ ] 水管走顶（要求走顶！漏水能早发现）
- [ ] 冷热水管都用热水管（壁厚耐压），左热右冷
- [ ] 前置过滤器装水表后（自购也行，工长代买也行）

### 水电验收（你在场才能验收）
- [ ] **打压测试**：0.8MPa 保 30 分钟，压降 ≤0.05MPa
- [ ] **全屋拍照+录像**，标尺量出管线距墙距离（以后打孔不打破）
- [ ] 检查每个插座是否通电、接线正确

---

## 三、第三阶段：地暖打压

- [ ] 收房时开发商地暖已铺设，装修期间保持压力表可见
- [ ] 如分水器移位或重新铺地暖，打压 0.6-0.8MPa 保压
- [ ] 回填层如果是豆石混凝土，养护 7 天

---

## 四、第四阶段：防水工程（2-3 天 + 闭水 48h）

- [ ] 卫生间：淋浴区墙面防水到 1.8m，地面+墙面至少 2 遍
- [ ] 卫生间门口外侧外延 50cm（防门口墙面返潮）
- [ ] 管根、墙角做 R 角圆弧 + 加强层
- [ ] **闭水试验 48 小时**，你亲自去楼下邻居家看天花板

---

## 五、第五阶段：瓦工贴砖（7-10 天）

### 瓷砖进场时你检查
- [ ] 核对品牌、型号、色号是否与合同清单一致
- [ ] 检查瓷砖有无破损、色差

### 铺贴巡查
- [ ] 客餐厅通铺用 750×1500mm 或 600×1200mm（问工长报价含哪种，大砖通常需加钱）
- [ ] 墙地对缝（提前跟瓦工说，不说默认不对缝）
- [ ] 淋浴区地面坡度 2-3%，倒水测试水全流向地漏
- [ ] 地漏做回字形切割

### 瓦工验收
- [ ] 空鼓锤逐块敲（全包最容易在瓦工偷工）
- [ ] 2 米靠尺平整度 ≤3mm
- [ ] 阴阳角方正 ≤3mm

---

## 六、第六阶段：木工吊顶（5-7 天）

- [ ] 轻钢龙骨（不用木龙骨）
- [ ] 窗帘盒深度 ≥15cm（电动窗帘 ≥20cm），宽度双轨 ≥25cm
- [ ] 石膏板拐角 L 型整板切割（防开裂）
- [ ] 如有中央空调/风管机，此时需已安装好室内机

---

## 七、第七阶段：油工（7-10 天）

- [ ] 确认用的腻子品牌（耐水腻子，郑州常见：美巢、立邦等）
- [ ] 确认乳胶漆品牌和色号
- [ ] 线槽、新旧墙交接处必须挂网
- [ ] 必须刷底漆（有些工长会省这一步）
- [ ] 强光手电侧面照验收，无波浪纹

---

## 八、第八阶段：安装季（5-7 天）

全包一般包含安装：厨卫吊顶、橱柜、室内门、开关插座、卫浴洁具。
需确认是否包含：全屋定制柜体安装、灯具安装。

### 安装顺序
1. 厨卫集成吊顶
2. 橱柜（烟灶同天协调）
3. 室内门 + 踢脚线
4. 全屋定制柜体
5. 开关插座 + 灯具
6. 卫浴（马桶、花洒、浴室柜）
7. 木地板（如有）
8. 美缝（砖贴完就可以做，不用等到最后）
9. 家电进场

---

## 九、全包容易出问题的地方（重点盯）

| 问题 | 怎么防 |
|------|--------|
| 瓷砖偷换品牌/等级 | 进场时核对包装箱 |
| 电线水管用非标线 | 剥开一段看铜芯颜色和直径 |
| 防水只刷一遍 | 刷的时候你在场看 |
| 腻子不用耐水的 | 看腻子袋子品牌 |
| 底漆被省略 | 底漆桶要看空桶 |
| 柜体板材用差的 | 看板材截面和环保证书 |
| 施工中途说"这个不在合同里要加钱" | 合同清单写清楚前别开工 |

---

## 十、工期与节点

- 全屋硬装正常工期：2.5-3 个月
- 定制柜体下单到安装：30-45 天（瓦工贴完砖立刻量尺下单）
- 室内门下单到安装：25-35 天（瓦工完就量尺）
- 不要赶在春节前 1 个月开工（工人急着回家，质量容易出问题）
