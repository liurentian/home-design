# 全屋家电选购清单（20 万预算版）

## 选购原则

- **不是所有电器都一次买齐**。入住必需品先买，提升型后买。
- 大促节点：618、双 11、双 12，能等的等到大促省 15-25%
- 线上比线下便宜，京东自营价保 30 天
- 能效：空调/冰箱必须一级能效（长期开，电费差很多）
- **先定需要嵌入/安装的电器**（洗碗机、烟灶、风管机），其余可入住后慢慢添

---

## 优先级说明

| 优先级 | 说明 |
|--------|------|
| 必买 | 入住前必须到位，否则影响正常生活 |
| 建议买 | 强烈建议，但可以入住后再装 |
| 可选 | 提升品质，预算有余再买 |

## 一、厨房电器

### 烟机 + 灶具（必买）
| 参数 | 建议 |
|------|------|
| 类型 | 侧吸（不碰头） |
| 风量 | ≥20m³/min |
| 风压 | ≥400Pa（高层尤其重要） |
| 灶具火力 | ≥5.0kW |
| 品牌 | 华帝/美的/万和（方太老板好但贵） |
| 预算 | ¥3000-4500 |
