# 验房指南

## 验房速查卡

**原则**：先验房，后签字。整改完成前不签收房文件。

**时机**：开发商通知收房后，确认已取得《竣工验收备案表》《住宅质量保证书》《住宅使用说明书》再验。

**人员**：
- 推荐请专业验房师（郑州 ¥350-580，116㎡），或让装修公司免费验
- 开发商工程人员会陪同，你在场盯

**标记方式**：每发现一处问题→贴便利贴编号→拍照→记录本上写清（房间+位置+描述）

---

## 问题严重等级

| 等级 | 标记 | 含义 | 处理方式 |
|------|------|------|---------|
| 🔴 严重 | 红标 | 影响安全和正常使用 | 必须开发商整改，不改不收房 |
| 🟡 一般 | 黄标 | 影响美观和使用体验 | 要求开发商整改，可在收房后限期改 |
| 🟢 轻微 | 绿标 | 装修时可覆盖 | 记录在案，装修时一并处理 |

---

## 验房走线顺序

按进门后的实际走动路线，避免漏查：

```
入户门 → 玄关 → 客餐厅 → 阳台 → 厨房 → 公卫 → 次卧1 → 次卧2/书房 → 主卧 → 主卫
```

每个空间按：顶面→墙面→地面→门窗→水电 的顺序查。

---

## 一、入户门

- [ ] 门框方正牢固，无变形 🔴
- [ ] 门扇开关顺畅无异响，门缝均匀（≤3mm）🟡
- [ ] 密封条完好贴合 🟡
- [ ] 锁具顺畅，反锁正常 🔴
- [ ] 猫眼安装到位、清晰 🟡
- [ ] 门扇表面无划痕磕碰 🟢
- [ ] 门槛无形变 🟡
- [ ] 防火门需查防火密封条和闭门器 🔴

> **关联施工**：入户门一般不换只换锁（智能锁），所以门本身的问题要在验房阶段解决。

---

## 二、全屋窗户（逐个查，每个窗都过一遍）

- [ ] 窗框固定牢固无松动 🔴
- [ ] 玻璃 3C 认证钢化，无划痕气泡裂纹 🔴
- [ ] 开关/推拉顺畅，把手灵活 🟡
- [ ] **A4 纸夹住抽拉测试密封性**（抽不出=合格）🟡
- [ ] 窗框与墙体接缝密封胶完好，无渗水痕迹 🔴
- [ ] 纱窗完好到位 🟡
- [ ] 窗台无裂缝渗水 🟡
- [ ] 飘窗台面平整无空鼓 🟡

### 阳台推拉门
- [ ] 轨道平直，滑动顺畅 🟡
- [ ] 门下排水孔是否通畅 🔴
- [ ] 锁扣正常，玻璃为安全玻璃 🔴

> **关联施工**：窗户密封不好会影响地暖保温和空调效果。如果后期要换窗或封阳台，需提前和工作室确认是否包含。

---

## 三、墙面与顶面（全屋通用，逐个空间查）

### 空鼓（验房第一重点）
- [ ] 空鼓锤逐面墙敲击，听到"咚咚"清脆声=空鼓 🔴
- [ ] 国家标准：单块 ≤400cm²（20×20cm），每自然间 ≤2 处
- [ ] **厨卫墙面空鼓必须整改**（贴砖后会脱落，全包施工方也会要求你先处理）
- [ ] 超出标准的空鼓，当场标记、拍照，要求开发商切开重新粉刷

### 裂缝
- [ ] 强光手电从侧面照射找裂缝
- [ ] 梁柱与填充墙交接处 — 最容易裂 🟡
- [ ] 线管开槽处 — 装修挂网可处理 🟢
- [ ] **门窗洞口四角 45°斜裂缝 — 结构问题信号** 🔴
- [ ] **贯穿性裂缝（墙两面都能看到）** 🔴
- [ ] 卫生间外墙贯穿裂缝 — 后期必渗水 🔴

### 平整度
- [ ] 2m 靠尺查墙面平整度（≤4mm）和垂直度（≤4mm）🟡
- [ ] 阴阳角是否为直角（A4 纸折角比对即可）🟡
- [ ] 顶面平整度，尤其吊顶位置 🟡

### 其他
- [ ] 墙面是否有水渍、霉斑 → 有渗漏 🔴
- [ ] 钢筋外露、铁丝外露 🔴
- [ ] 明显修补痕迹 → 可能掩盖问题 🟡

> **关联施工**：空鼓和裂缝不处理好，全包装修后油工会推脱"是你墙面基础问题"。务必在验房阶段解决，墙面的锅不能甩给施工方。

---

## 四、地面检查（全屋）

- [ ] 贯穿性裂缝 🔴
- [ ] 空鼓（地暖回填层空鼓影响铺砖）🔴
- [ ] 平整度 2m 靠尺 ≤4mm 🟡
- [ ] **各房间地面与公共走廊的高差** — 影响找平厚度和成本 🟡
- [ ] 起砂、起灰 🟡
- [ ] 水渍、潮湿 🟡

> **关联施工**：地面高差过大会增加找平费用。如果全包按面积一口价，工长可能会拿这个说事加钱。验房时心里有个数。

---

## 五、水电检查

### 强电
- [ ] 相位检测仪逐个插座：左零右火上接地 🔴
- [ ] 配电箱回路标识清晰、漏保正常、总开关正常 🔴
- [ ] 电线规格：照明 1.5mm² / 普通插座 2.5mm² / 厨房空调 4mm² 🟡
- [ ] 弱电箱：光纤、网线、电视线是否入户 🟡
- [ ] 对讲系统正常 🟡

### 给排水
- [ ] 开水龙头查水压是否正常 🔴
- [ ] 倒水测每个地漏排水是否顺畅 🔴
- [ ] 台盆、马桶下水管通畅 🟡
- [ ] 排水管是否有存水弯（防臭）🟡
- [ ] 下水管检修口是否密封 🟡

### 防水（别省！）
- [ ] 卫生间顶面是否有渗漏痕迹 🔴
- [ ] **闭水试验**：卫生间蓄水 2-3cm，48h 后去楼下邻居家看天花板 🔴
- [ ] 阳台地面防水 🔴
- [ ] 厨房防水（如有地漏）🟡

> **关联施工**：闭水试验的结果直接影响全包施工的防水方案。如果开发商防水没做好，要求他们先整改再做装修防水，否则互相推责任。

---

## 六、厨卫专项

### 厨房
- [ ] 烟道止逆阀是否安装、密封 🟡
- [ ] 燃气表位置是否合理（影响橱柜布局）🟡
- [ ] 燃气管道有无腐蚀 🔴
- [ ] 排烟孔位置和孔径 🟡

### 卫生间
- [ ] 下沉式卫生间回填层有无裂缝 🔴
- [ ] 排污管、排水管位置与户型图一致 🟡
- [ ] **等电位端子箱是否安装**（安全关键，装修别封死）🔴
- [ ] 通风管道通畅 🟡

> **关联施工**：燃气表位置直接决定橱柜方案。如果位置不合理，需要联系燃气公司移位（装修前搞定）。

---

## 七、测量复核

| 项目 | 标准 | 实测记录 |
|------|------|---------|
| 客厅层高 | 与合同一致，四角偏差 ≤15mm | |
| 主卧层高 | 同上 | |
| 次卧1层高 | 同上 | |
| 书房层高 | 同上 | |
| 客厅开间×进深 | 偏差 ≤10mm | |
| 主卧开间×进深 | 偏差 ≤10mm | |
| 次卧1开间×进深 | 偏差 ≤10mm | |
| 书房开间×进深 | 偏差 ≤10mm | |
| 各房间对角线差 | ≤20mm | |

> **关联施工**：对角线偏差过大会影响瓷砖铺贴的方正度。实测数据交给全包工长，让他们心里有数。

---

## 八、暖通检查

- [ ] 地暖分水器位置 🔴
- [ ] **地暖打压**：0.6-0.8MPa 保 24h，压降 ≤0.05MPa 🔴
- [ ] 空调孔位置和内高外低坡度 🔴
- [ ] 新风系统（如有）正常运转 🟡

> **关联施工**：地暖打压有问题的，装修期间保持压力表可视。全包施工期间如果动了地暖分水器，打压记录就是责任分界线。

---

## 九、验房收尾

### 问题汇总与整改
- 与开发商工程人员逐项确认所有标记的问题
- 出具书面整改通知单，双方签字
- 约定整改完成时间和复验时间
- 🔴 项整改完成前不签字收房
- 重大结构问题可拒绝收房

### 顺手做的事
- 验房当天同步约装修公司/工作室来量房（一趟搞定）
- 把全屋照片和视频存档，尤其水电点位、管道位置
- 记下强电箱、弱电箱、分水器的精确位置和尺寸

### 郑州本地验房资源
- 专业验房：牛角监、土巴兔验收、装小蜜
- 装修公司免费验房（量房+验房同天，最省事）
- 小区业主群拼团验房（人多有折扣）
