# 验房指南

## 一、准备阶段

### 验房时机
- 开发商通知收房后，先验房再签字收房
- 验房前确认开发商已取得《竣工验收备案表》《住宅质量保证书》《住宅使用说明书》

### 自带工具
| 工具 | 用途 |
|------|------|
| 空鼓锤（伸缩款） | 检查墙面/地面空鼓 |
| 水平仪 / 2米靠尺 | 检查墙面地面平整度、垂直度 |
| 卷尺（5米） | 测量层高、开间进深、对角线 |
| 相位检测仪（验电器） | 检查插座接线是否正确 |
| 强光手电筒 | 检查墙面裂缝、阴阳角 |
| 水桶 / 水管 | 检查地漏排水、下水管道通畅 |
| 粉笔 / 便利贴 | 标记问题点 |
| A4 纸 | 检查窗户密封性 |
| 手机 / 相机 | 拍照存档 |

### 验房陪同
- 开发商一般会派工程人员陪同
- 建议花钱请专业验房师（郑州行情约 3-5 元/㎡，116㎡ 约 ¥350-580）
- 如果找装修公司，部分公司提供免费验房服务，可提前联系

## 二、入户门检查

- [ ] 门框安装是否牢固、方正，有无变形
- [ ] 门扇开关是否顺畅，无异响
- [ ] 门扇与门框缝隙是否均匀（≤3mm）
- [ ] 密封条是否完好、贴合
- [ ] 锁具开合是否顺畅，反锁功能是否正常
- [ ] 猫眼是否安装、清晰度
- [ ] 门扇表面有无划痕、磕碰、色差
- [ ] 门槛是否有变形、划痕
- [ ] **如果是防火门**：检查防火密封条和闭门器

---

## 三、门窗检查（所有房间）

### 窗户（每个窗户逐项查）
- [ ] 窗框固定是否牢固，有无松动
- [ ] 玻璃是否有划痕、气泡、裂纹
- [ ] 玻璃是否为 **3C 认证钢化玻璃**（看角落是否有 3C 标志）
- [ ] 窗户推拉/平开是否顺畅，把手转动是否灵活
- [ ] 关窗后用 A4 纸夹住抽拉，检查密封性（抽不出为合格）
- [ ] 窗框与墙体接缝是否有密封胶，有无渗水痕迹
- [ ] 纱窗是否完好、安装到位
- [ ] 窗台是否有裂缝、渗水痕迹
- [ ] 飘窗台面是否平整、无空鼓

### 阳台推拉门
- [ ] 轨道是否平直，滑动是否顺畅
- [ ] 门下是否有排水孔
- [ ] 锁扣是否正常
- [ ] 玻璃是否为安全玻璃

---

## 四、墙面与顶面检查

### 空鼓检查（重点！）
- [ ] 用空鼓锤逐面墙、逐块地砖敲击
- [ ] 听到 **"咚咚"清脆声** 即为空鼓
- [ ] 国家标准：单块空鼓面积 ≤400cm²（20cm×20cm），且每自然间不超过 2 处
- [ ] **空鼓超过标准必须要求开发商整改**（切开重新粉刷/铺贴）
- [ ] 厨卫墙面空鼓尤其重要，贴砖后会脱落

### 裂缝检查
- [ ] 强光手电照射墙面，检查有无裂缝
- [ ] 重点检查：
  - 梁柱与填充墙交接处（最容易开裂）
  - 线管开槽处
  - 门窗洞口四角（45°斜裂缝说明结构问题）
- [ ] 细微龟裂纹 → 装修时挂网处理即可
- [ ] 贯穿性裂缝 → 需开发商处理，特别是卫生间外墙（会渗水）

### 平整度与垂直度
- [ ] 用 2 米靠尺检查墙面平整度（偏差 ≤4mm）
- [ ] 检查墙面垂直度（偏差 ≤4mm）
- [ ] 检查阴阳角是否为直角（用直角尺或 A4 纸折角比对）
- [ ] 检查顶面平整度，特别是吊顶位置

### 其他
- [ ] 墙面是否有水渍、霉斑（说明有渗漏）
- [ ] 墙面是否有钢筋外露、铁丝外露
- [ ] 墙面是否有明显修补痕迹（可能掩盖问题）

---

## 五、地面检查

- [ ] 地面是否有裂缝（特别是贯穿性裂缝）
- [ ] 地面是否有空鼓（地暖回填层空鼓影响后期铺砖）
- [ ] 地面平整度（2米靠尺偏差 ≤4mm）
- [ ] **各房间地面与公共走廊是否有明显高差**（影响后期找平厚度）
- [ ] 地面是否有起砂、起灰现象
- [ ] 检查地面是否有水渍、潮湿痕迹

---

## 六、水电管路检查

### 强电
- [ ] 用电笔/相位检测仪检查每个插座：
  - 接线是否正确（左零右火上接地）
  - 是否有电
  - 漏电保护是否有效
- [ ] 检查配电箱：
  - 各回路标识是否清晰
  - 总开关、漏保、分路开关是否正常
  - 电线规格：照明 1.5mm²、普通插座 2.5mm²、厨房/空调 4mm²
- [ ] 检查弱电箱：网线、电视线、光纤是否入户
- [ ] 检查对讲系统是否正常

### 给排水
- [ ] 打开所有水龙头，检查水流是否正常、有无抖动
- [ ] 检查水表是否正常运转
- [ ] 检查每个地漏排水是否顺畅（倒水测试）
- [ ] 检查台盆、马桶下水管是否通畅
- [ ] 检查排水管是否有 S 弯/存水弯（防臭）
- [ ] 检查下水管是否有检修口、是否密封

### 防水（关键！）
- [ ] 卫生间顶面是否有水渍、渗漏痕迹
- [ ] **建议做闭水试验**：在卫生间地面蓄水 2-3cm，24-48 小时后去楼下邻居家检查天花板是否有渗漏
- [ ] 阳台地面防水检查
- [ ] 厨房防水检查（如有地漏）

---

## 七、厨卫专项检查

### 厨房
- [ ] 烟道止逆阀是否安装、密封
- [ ] 燃气表位置是否合理（影响后期橱柜设计）
- [ ] 燃气管道是否有腐蚀
- [ ] 排烟孔位置、孔径

### 卫生间
- [ ] 下沉式卫生间回填层是否有裂缝
- [ ] 排污管、排水管位置是否与户型图一致
- [ ] 等电位端子箱是否安装（安全关键！装修时不要封死）
- [ ] 通风管道是否通畅

---

## 八、测量复核

### 层高
- [ ] 各房间层高是否与合同一致（通常 2.8-3.0m）
- [ ] 同一房间四角层高偏差 ≤15mm

### 开间与进深
- [ ] 客厅开间 × 进深
- [ ] 主卧开间 × 进深
- [ ] 次卧 1 开间 × 进深
- [ ] 次卧 2/书房开间 × 进深
- [ ] 与合同户型图标注尺寸偏差 ≤10mm

### 方正度
- [ ] 测量每个房间对角线，两条对角线长度差 ≤20mm
- [ ] 对角线偏差过大会影响家具摆放和瓷砖铺贴

---

## 九、暖通检查

- [ ] 地暖/暖气片分水器位置、管路标识
- [ ] **地暖必须做打压测试**：保持 0.6-0.8MPa 压力 24 小时，压降 ≤0.05MPa 为合格
- [ ] 检查空调孔位置是否合理、是否有坡度（内高外低防雨水倒灌）
- [ ] 新风系统（如有）是否运转正常

---

## 十、验房报告与整改

### 问题记录
- 每发现一个问题，用便利贴编号贴在旁边，同时拍照
- 记录本上写下：房间 + 问题位置 + 问题描述
- 验房结束后与开发商工程人员逐项确认

### 整改要求
- 出具书面整改通知单，双方签字确认
- 约定整改完成时间和复验时间
- **整改完成前不要签字收房**
- 重大结构问题（贯穿裂缝、严重渗漏）可拒绝收房

### 郑州本地参考
- 专业验房公司：牛角监、土巴兔验收、装小蜜
- 验房 + 量房可以同一天完成（装修公司通常免费量房）
