# 验房指南

## 验房速查卡

**原则**：先验房，后签字。整改完成前不签收房文件。

**时机**：开发商通知收房后，确认已取得《竣工验收备案表》《住宅质量保证书》《住宅使用说明书》再验。

**人员**：
- 推荐请专业验房师（郑州 ¥350-580，116㎡），或让装修公司免费验
- 开发商工程人员会陪同，你在场盯

**当天流程**（约 2-2.5 小时）：
```
到现场 → 按走线逐间检查（约 1.5h）→ 汇总问题贴标拍照（15min）→ 与工程人员逐项确认签字（20min）
```
注意：去之前联系楼下邻居，确认是否可以配合查渗漏。

**标记方式**：每发现一处问题→贴便利贴编号→拍照→记录本上写清（房间+位置+描述）

---

## 问题严重等级

| 等级 | 标记 | 含义 | 处理方式 |
|------|------|------|---------|
| 🔴 严重 | 红标 | 影响安全和正常使用 | 必须开发商整改，不改不收房 |
| 🟡 一般 | 黄标 | 影响美观和使用体验 | 要求开发商整改，可在收房后限期改 |
| 🟢 轻微 | 绿标 | 装修时可覆盖 | 记录在案，装修时一并处理 |

---

## 验房走线顺序

按进门后的实际走动路线，避免漏查：

```
入户门 → 入户花园 → 客餐厅 → 阳台 → 厨房 → 公卫 → 南次卧 → 北次卧 → 主卧 → 主卫
```

每个空间按：顶面→墙面→地面→门窗→水电 的顺序查。

## 一、入户门

- [ ] 推拉门扇，看是否顺畅、有无异响 🔴
- [ ] 摇晃门扇，检查门框是否牢固 🔴
- [ ] 用卷尺量门扇与门框缝隙，上下左右四处，≤3mm 为合格 🟡
- [ ] 看密封条是否完整贴合，手摸无脱落 🟡
- [ ] 反复开关锁具，反锁测试是否顺畅 🔴
- [ ] 通过猫眼向外看，是否清晰无遮挡 🟡
- [ ] 看门扇表面有无划痕磕碰、门槛有无变形 🟢
- [ ] 防火门判断：看门框侧面有无防火标签/铭牌（一般入户门都是防火门），有的话检查防火密封条和闭门器是否到位 🔴

> **关联施工**：入户门一般不换只换智能锁，门本身的问题验房阶段解决。

---

## 二、全屋窗户（每个窗都过一遍）

- [ ] 用手推拉窗框，检查是否牢固无松动 🔴
- [ ] 看玻璃角落有无 **3C 标志**（钢化认证），找划痕、气泡、裂纹 🔴
- [ ] 开关/推拉窗扇，检查是否顺畅，把手转动是否灵活 🟡
- [ ] **A4 纸密封测试**：关窗，A4 纸夹在窗扇与窗框之间，用力抽拉。抽不出=密封合格 🟡
- [ ] 看窗框与墙体接缝处的密封胶是否完整，有无开裂脱落；窗台下方墙面有无水渍 🔴
- [ ] 纱窗是否完好、能正常开关 🟡
- [ ] 看窗台有无裂缝、飘窗台面敲击听空鼓 🟡

### 阳台推拉门
- [ ] 来回推拉，看轨道是否平直顺畅 🟡
- [ ] 蹲下看门下是否有排水孔、孔是否堵塞 🔴
- [ ] 测试锁扣是否正常、看玻璃是否为安全玻璃（3C 标志）🔴

> **关联施工**：窗户密封不好影响地暖保温和空调效果。后期换窗或封阳台需提前确认是否在全包范围内。

---

## 三、墙面与顶面（全屋通用，逐空间查）

### 空鼓 — 验房第一重点

**怎么敲**：用空鼓锤（或螺丝刀柄）轻敲墙面，匀速移动。实墙声音沉闷"哒哒"，空鼓声音清脆"咚咚"，区别很明显。

- [ ] 逐面墙敲击，听到清脆声即用便利贴标记位置 🔴
- [ ] 面积参考：巴掌大小（约 20×20cm）以内的空鼓可接受，超过需整改。具体面积让开发商人员现场测量确认，你只管标位置 🔴
- [ ] **厨卫墙面空鼓优先处理**（此处以后要贴砖，空鼓必脱落）🔴
- [ ] 地面同样敲击检查空鼓（不区分回填层还是结构层，有空洞声就标）🔴

### 裂缝

**怎么找裂缝**：打开手机闪光灯，从侧面（不是正面）贴着墙面照，裂缝会投下阴影。

三种裂缝的识别与处理：

| 裂缝类型 | 外观 | 处理 |
|----------|------|------|
| 龟裂纹 | 像干涸的泥地，细密网状 | 🟢 装修挂网即可 |
| 竖向/水平直线裂缝 | 通常在梁柱与填充墙交接处、线管开槽处 | 🟡 装修挂网处理 |
| **45°斜裂缝** | 从门窗洞口角向外延伸约 45°，像对角线方向 | 🔴 结构问题信号，必须开发商处理 |

- [ ] 强光手电侧照全屋墙面找裂缝 🔴
- [ ] 重点关注门窗洞口四角有无 45°斜裂缝 🔴
- [ ] 用手摸裂缝判断宽度：有明显凹凸感或信用卡边缘能塞进去（约 >0.3mm）→ 需开发商处理 🔴

### 平整度（没靠尺怎么办）

- [ ] 手机装水平仪 App，贴在墙上可以看倾斜度 🟡
- [ ] 或找一根长条木方/铝合金踢脚线，贴墙看有没有明显缝隙 🟡
- [ ] 阴阳角用 A4 纸折直角比对，看是否贴合 🟡
- [ ] 顶面抬头看，手电筒斜照有没有明显凹凸波浪即可 🟡

### 其他墙面问题
- [ ] 看墙面有无水渍印、霉斑 → 有渗漏 🔴
- [ ] 看有无钢筋头外露、铁丝网外露 🔴
- [ ] 看有无明显修补痕迹（一块颜色不同）→ 可能掩盖问题 🟡

> **关联施工**：空鼓裂缝不处理，全包油工会说"墙面基础问题"不背锅。验房阶段解决，责任分清楚。

---

## 四、地面检查（全屋）

- [ ] 看地面有无明显裂缝，裂缝宽度参照墙面裂缝判断标准 🔴
- [ ] 空鼓锤敲击地面，有空洞声就标（不区分哪层空，统一标）🔴
- [ ] 手机水平仪或长木条查平整度 🟡
- [ ] **用卷尺量各房间地面与公共走廊的高差**（影响后期找平厚度和费用）🟡
- [ ] 脚在地面来回蹭，看是否起砂、起灰粉 🟡
- [ ] 看地面有无水渍、返潮 🟡

> **关联施工**：地面高差过大会增加找平成本，工长可能借机加钱。验房时记下数据心里有数。

---

## 五、水电检查

### 强电
- [ ] 相位检测仪逐个插每个插座，面板指示灯会显示接线是否正确（左零右火上接地）🔴
- [ ] 按配电箱内每个开关的"测试"按钮，漏保应自动跳闸，复位后能合上 🔴
- [ ] 看配电箱回路标签是否贴好、字迹清楚 🟡
- [ ] 打开弱电箱门，看光纤、网线、电视线有没有穿到箱内 🟡
- [ ] 试一下对讲系统是否能接通 🟡

### 给排水
- [ ] 所有水龙头开到最大，看出水是否有力、无抖动 🔴
- [ ] 每个地漏倒一瓶水，看水是否迅速流走、无积水 🔴
- [ ] 马桶、台盆冲水是否通畅 🟡
- [ ] 下水管接口看一圈，有无明显缝隙或漏水痕迹 🟡

### 防水（重点）
- [ ] 抬头看卫生间天花板，尤其管道根部周围，有无水渍、墙皮起泡 🔴
- [ ] 看阳台地面防水层是否完好 🔴
- [ ] **闭水试验**：这是验房最重要的一项，但当天做不了 48 小时。操作方式：
  1. 先用塑料袋装沙土或旧毛巾，把地漏塞紧
  2. 蓄水 2-3cm 深
  3. 拍一张水位照片记录时间
  4. 24 小时后联系楼下邻居查看天花板有无渗漏
  5. 如果楼下暂无人入住，找物业拿钥匙协调

> **关联施工**：闭水试验结果决定全包防水方案。开发商防水有问题→先让他们整改→再做装修防水，责任不交叉。

## 六、暖通检查

- [ ] 找到地暖分水器（一般在水井房或厨房/阳台墙面，一个金属箱体带多个阀门），拍照记录位置 🔴
- [ ] 看分水器上的压力表是否有读数。开发商交房时一般会带压交付（0.2-0.4MPa 左右），如果归零说明管道可能有问题 🔴
- [ ] 看每个空调孔洞：孔内壁是否光滑、孔是否内高外低（从里面往外看略微向下倾斜，防雨水倒灌）🔴
- [ ] 如有新风系统，开机试运转听有无异响 🟡

> **关联施工**：地暖压力表装修期间保持可视，不要遮挡。施工方一旦动了分水器，压力变化就是责任分界线。

---

## 七、厨房专项

- [ ] 看烟道口止逆阀是否已安装、密封圈是否贴合 🟡
- [ ] 看燃气管道是否有锈蚀、腐蚀痕迹 🔴
- [ ] **拍照记录燃气表精确位置**和周围尺寸（卷尺量），供后续橱柜设计使用。不要自己判断"合不合理"，你还没定橱柜方案 🟡
- [ ] 看排烟孔位置和孔径大小 🟡

---

## 八、卫生间专项

- [ ] 看排污管、排水管位置与户型图标注是否一致 🟡
- [ ] **找等电位端子箱**：一般在洗手台下方墙面，是一个带"等电位"字样或接地标志⚡的小金属盒子，巴掌大小。确认有，且装修时不要封死 🔴
- [ ] 打开排气扇看通风管道是否通畅（用纸巾测试：开机后纸巾能被吸住=通畅）🟡

|------|------|---------|
| 客厅层高 | 合同一致，四角偏差 ≤15mm | |
| 主卧层高 | 同上 | |
| 南次卧层高 | 同上 | |
| 北次卧层高 | 同上 | |
| 客厅开间×进深 | 偏差 ≤10mm | |
| 主卧开间×进深 | 偏差 ≤10mm | |
| 南次卧开间×进深 | 偏差 ≤10mm | |
| 北次卧开间×进深 | 偏差 ≤10mm | |

> **关联施工**：对角线偏差过大会影响瓷砖铺贴方正度。实测数据交给工长，让他们提前知道。

## 十、验房收尾

### 问题汇总与整改
- 与开发商工程人员逐项确认所有标记的问题
- 出具书面整改通知单，双方签字
- 约定整改完成时间和复验时间
- 🔴 项整改完成前不签字收房
- 重大结构问题（贯穿大裂缝、严重渗漏）可拒绝收房

### 顺手做的事
- 验房当天同步约工作室来量房（一趟搞定）
- 把全屋照片和视频存档，尤其水电点位和管道位置
- 记下强电箱、弱电箱、分水器的精确位置和尺寸

### 郑州本地验房资源
- 专业验房：牛角监、土巴兔验收、装小蜜
- 装修公司免费验房（量房+验房同天，最省事）
- 小区业主群拼团验房（人多有折扣）
