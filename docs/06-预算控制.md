## 三、预算分配策略（钱花在刀刃上）

### 不能省
- 水电材料品牌（要国标，不要非标线）
- 防水材料（要雨虹/德高/西卡等品牌）
- 乳胶漆（立邦/多乐士/三棵树，要正品不要工程漆）
- 开关插座面板（公牛/德力西即可，不用进口品牌）
- 每天高频使用的东西：床垫、沙发、烟机、冰箱

### 可以降档/后买
- 瓷砖不追广东大品牌，广东产地的合格砖就行
- 木地板用强化复合代替实木多层/实木（便宜的不好看，但够用）
- 室内门用免漆门代替烤漆门（差 500-800/扇）
- 定制柜体用本地工厂代替品牌店（差 30-50%）
- 次卧空调、电动窗帘、扫拖机器人可入住后买
- 电视可入住后买（投影先用手机也行）

## 四、全包合同避坑指南

### 材料替换
- 合同中写明："所有主材进场时通知甲方核对品牌型号"
- 全包最常见的坑：展厅看的品牌好，送到家的同品牌低端系列
- 解决：合同写清楚品牌+系列+型号，不要只写品牌

### 增项陷阱
- 签合同前问清楚："这个报价里有没有不含的？"
- 常见增项：水电超点位、大砖铺贴费、墙固地固、成品保护费、垃圾外运费
- 在合同上写：**"本合同总价包含以上所有项目，除甲方主动新增项目外，不再产生额外费用"**

### 亲人工作室的议价话术
- "咱们是亲人，价格上我不跟你砍，但材料和工艺你按自己家装修的标准来"
- 不要一上来就"越便宜越好"，工长听了就敢用差材料
- 直接说：**"材料用好一点，总共控制在 X 万就行"**

## 六、超支预警

预计最容易超支的地方：
1. **全屋定制柜体超预期**（如果全包不含，这 3-4 万是额外的）
2. **水电增项**（超过约定点位数量）
3. **看到好东西想升级**（逛一次建材市场就想升级三样）
4. **家具超预算**（一张床垫 3000 变 8000）

对策：
- 柜体超支 → 先做必须的（主卧衣柜+鞋柜+餐边柜），其他后期加
- 某类花多了 → 从备用金走，同时在别处省回来
