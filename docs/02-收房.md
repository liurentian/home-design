# 收房指南

## 一、收房前置条件

### 开发商必须提供的文件（三书一表一证）
| 文件 | 说明 |
|------|------|
| 《住宅质量保证书》 | 开发商对房屋质量的承诺，含保修范围及期限 |
| 《住宅使用说明书》 | 房屋使用说明及注意事项 |
| 《建筑工程竣工验收备案表》 | 建委备案章必须齐全，无此表不得交付 |
| 《面积实测报告》 | 房管局测绘队出具的实测面积 |
| 《房地产开发建设项目竣工综合验收合格证》 | 综合验收通过证明 |

### 你的权利
- **先验房、后收房**。开发商要求先签字交钱再验房是违规的
- 验房发现的问��，开发商必须在约定时间内整改完毕
- 重大质量问题（主体结构、严重渗漏等）可拒绝收房

---

## 二、面积核对

### 面积差异处理（合同一般会约定）
根据《商品房销售管理办法》第二十条：
- 面积误差 ≤3%：按合同单价多退少补
- 面积误差 >3%：买方有权退房
  - 不退房的：超出 3% 部分由开发商承担，产权归买方
  - 面积缩水超 3%：超出 3% 部分双倍返还

### 你需要做的
- [ ] 核对《面积实测报告》与合同约定面积
- [ ] 自行测量各空间面积，与实测报告大致吻合即可
- [ ] 若面积差异明显，委托第三方测绘复核

## 三、费用结算

### 收房时可能涉及的费用
| 费用项目 | 说明 | 预估金额（郑州） |
|----------|------|------------------|
| 面积补差款 | 实测面积与合同面积差异的补/退款 | 视情况 |
| 契税 | 140㎡以下首套 1% | 按总价计算 |
| 维修基金 | 专项用于公共部位维修 | 郑州约 65 元/㎡ |
| 物业费 | 预缴 1 年，3.18 元/㎡/月 | 116㎡ × 3.18 × 12 ≈ ¥4,427 |
| 装修保证金 | 装修验收后退还 | ¥3,000 |
| 装修垃圾清运费 | 10 元/㎡ | 116㎡ × 10 = ¥1,160 |
| 电费预缴 | 预存至电表账户 | ¥200 |

### 注意事项
- [ ] 契税可自己到税务局缴纳，不一定通过开发商代收
- [ ] 维修基金有专用账户，不要交给开发商个人
- [ ] 所有费用索取正规发票或收据
- [ ] 物业费从收房之日起算，提前确认收费标准

## 四、收房办理材料

### 业主本人办理
- [ ] 身份证原件 + **复印件 2 份**
- [ ] 购房合同原件

### 非业主本人办理（代收房）
- [ ] **委托书**（业主本人亲笔签名 + 按手印）
- [ ] 业主身份证复印件 2 份
- [ ] 代办人身份证原件 + 复印件 2 份
- [ ] 购房合同原件

> 委托书无固定格式，写明：业主姓名、身份证号、房号、代办人姓名、身份证号、委托事项（收房），签字按手印即可。

## 五、物业交割

### 钥匙交接
- [ ] 入户门钥匙（A/B 钥匙说明，装修钥匙会在装修钥匙启用后失效）
- [ ] 单元门禁卡/人脸录入
- [ ] 信报箱钥匙
- [ ] 水井、电井钥匙（有些小区由物业保管）

### 水电气暖
- [ ] 水表底数拍照记录
- [ ] 电表底数拍照记录
- [ ] 燃气表底数拍照记录（确认燃气已开通）
- [ ] 地暖/暖气确认打压状态

### 其他
- [ ] 签署《业主临时管理规约》
- [ ] 登记业主信息（用于业委会选举等）
- [ ] 了解物业报修流程和值班电话
- [ ] 加入小区业主群

## 六、郑州收房特别提示

### 本地政策
- 郑州自 2023 年起推行"交房即交证"，收房时确认不动产权证办理进度
- 部分区域要求装修前到物业报备图纸（水电改造图）

### 季节因素
- **冬季收房**（12-2 月）：检查采暖系统，确认地暖打压正常
- **雨季收房**（6-9 月）：重点检查渗漏，雨天收房反而更容易发现问题

---

## 七、收房后、装修前

收房完成后、正式开工前，还有几件事要做：

- [ ] **量房**：找 2-3 家装修公司免费量房出方案，对比报价
- [ ] **确定设计**：根据量房 CAD 图确定平面布局
- [ ] **封阳台**：如果需要封阳台，这是最早可以做的（室内装修前完成）
- [ ] **物业报备**：提交装修图纸、施工方资质、缴纳装修押金
- [ ] **确定装修模式**：自装 / 半包 / 全包 / 整装
