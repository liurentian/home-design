## 七、定位当天操作流程

1. **提前一晚**：打印这张表，按房间画好自己需要的插座和设备位置
2. **定位当天**：约工长+水电师傅，一个房间一个房间走
3. **你在前面说** → 师傅在墙上粉笔画框标位置 → 你确认 → 下一项
4. **全部画完后**：从头到尾走一遍，每个标记点用手机拍照
5. **晚上回去**：对着照片再核对一遍，有漏的第二天补
6. **补完后**：师傅按确认的标记开槽，不再改动
