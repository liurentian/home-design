一、入户门

  • 🔴 推拉门扇看是否顺畅、有无异响；摇晃门扇查门框是否牢固
  • 🔴 反复开关锁具，反锁测试
  • 卷尺量门缝上下左右四处，≤3mm 合格
  • 看密封条是否完整贴合；猫眼向外看是否清晰
  • 看门扇表面有无划痕磕碰、门槛有无变形
  • 🔴 防火门判断：看门框侧面有无防火标签/铭牌，一般入户门都是防火门。是的话查防火密封条和闭门器
关联施工：入户门一般不换只换智能锁，门本身的问题验房阶段解决。

二、全屋窗户（每个窗都过一遍）

  • 🔴 用手推拉窗框检查是否牢固；看玻璃角落有无 3C 标志，找划痕气泡裂纹
  • 🔴 看窗框与墙体接缝密封胶是否完整，窗台下墙面有无水渍
  • 开关/推拉窗扇查顺畅度，把手转动灵活
  • A4 纸密封测试：关窗后纸夹窗扇与窗框间用力抽拉，抽不出=合格
  • 纱窗是否完好能正常开关；窗台有无裂缝；飘窗台面敲击听空鼓
  • 🔴 阳台推拉门：来回推拉查轨道是否平直，蹲看门下排水孔是否堵塞，锁扣和3C玻璃
关联施工：窗户密封不好影响地暖保温和空调效果。后期换窗或封阳台需确认是否在全包范围内。

三、墙面与顶面（全屋逐空间查）

空鼓 — 验房第一重点

怎么敲：空鼓锤（或螺丝刀柄）轻敲墙面匀速移动。实墙沉闷"哒哒"，空鼓清脆"咚咚"，区别明显。巴掌大小（约20×20cm）以内可接受，超过需整改。具体面积让开发商人员测，你只管标位置。
厨卫墙面空鼓优先处理——以后要贴砖，空鼓必脱落。地面同样敲击，有空洞声就标，不区分层级。
  • 🔴 逐面墙敲击，清脆声处用便利贴标记

裂缝

怎么找：打开手机闪光灯，从侧面贴着墙面照，裂缝会投下阴影。
三种裂缝：① 龟裂纹（细密网状）→ 🟢装修挂网即可；② 竖向/水平直线裂缝（梁柱交接处、线管开槽处）→ 🟡装修挂网处理；③ 45°斜裂缝（从门窗洞口角向外延伸约45°）→ 🔴 结构问题信号，必须开发商处理
  • 🔴 强光手电侧照全屋墙面找裂缝
  • 🔴 门窗洞口四角有无45°斜裂缝
  • 🔴 手摸裂缝宽度：有明显凹凸感或信用卡边缘能塞进去（>0.3mm）→ 需开发商处理

平整度 + 其他

  • 手机水平仪 App 或长条木方贴墙查平整度；A4 纸折角比阴阳角直角
  • 顶面抬头看，手电筒斜照有无明显凹凸波浪
  • 🔴 看墙面有无水渍印、霉斑 → 有渗漏
  • 🔴 看有无钢筋头外露、铁丝网外露
  • 看有无明显修补痕迹（一块颜色不同）→ 可能掩盖问题
关联施工：空鼓裂缝不处理，油工会说"墙面基础问题"不背锅。验房阶段解决，责任分清。

四、地面检查（全屋）

  • 🔴 看地面有无明显裂缝，宽度判断：指甲刮有明显凹凸感或信用卡能塞进 → 需处理
  • 🔴 空鼓锤敲地面，有空洞声就标（不区分回填层还是结构层）
  • 手机水平仪或长木条查平整度
  • 卷尺量各房间地面与公共走廊高差 — 影响找平成本
  • 脚在地面来回蹭，看是否起砂起灰；看有无水渍返潮
关联施工：地面高差过大会增加找平费用，工长可能拿这个加钱。验房时记下数据心里有数。

五、水电检查

强电

  • 🔴 相位检测仪逐个插每个插座，看指示灯显示接线是否正确（左零右火上接地）
  • 🔴 按配电箱每个开关的"测试"按钮，漏保应自动跳闸，复位后能合上
  • 看配电箱回路标签是否贴好、字迹清楚
  • 打开弱电箱门，看光纤、网线、电视线有没有穿到箱内；试对讲系统

给排水

  • 🔴 所有水龙头开到最大，看出水是否有力；每个地漏倒一瓶水，看是否迅速流走
  • 马桶、台盆冲水是否通畅；下水管接口有无明显缝隙或漏水

防水 — 绝对不能省

  • 🔴 抬头看卫生间天花板，尤其管道根部周围，有无水渍、墙皮起泡
  • 🔴 看阳台地面防水层是否完好
  • 🔴 闭水试验：①塑料袋装沙土或旧毛巾塞紧地漏 ②蓄水2-3cm ③拍照记录水位和时间 ④24h后联系楼下邻居看天花板有无渗漏 ⑤楼下无人找物业拿钥匙
关联施工：闭水试验结果决定全包防水方案。开发商防水有问题→先让他们整改→再做装修防水，责任不交叉。

六、暖通检查

  • 🔴 找到地暖分水器（一般在水井房或厨房/阳台墙面，金属箱体带多个阀门），拍照记录位置
  • 🔴 看分水器压力表：开发商交房一般带压交付（0.2-0.4MPa左右），归零说明管道可能有问题
  • 🔴 看每个空调孔：内高外低（从里面往外看略微向下倾斜）防雨水倒灌
  • 如有新风系统，开机听有无异响
关联施工：地暖压力表装修期间保持可视。施工方动了分水器，压力变化就是责任分界线。

七、厨房专项

  • 看烟道口止逆阀是否已安装、密封圈是否贴合
  • 🔴 看燃气管道是否有锈蚀腐蚀痕迹
  • 拍照记录燃气表精确位置并用卷尺量周围尺寸，供橱柜设计用。不要自己判断"合不合理"，还没定橱柜方案
  • 看排烟孔位置和孔径

八、卫生间专项

  • 看排污管、排水管位置与户型图标注是否一致
  • 🔴 找等电位端子箱：一般在洗手台下方墙面，是一个带"等电位"字样或接地标志⚡的小金属盒子，巴掌大小。确认存在，装修时不要封死
  • 纸巾测试排气通风：开机后纸巾能被吸住=通畅

十、验房收尾

  • 与开发商工程人员逐项确认所有标记问题
  • 出具书面整改通知单，双方签字，约定整改完成时间和复验时间

主卧 L 形衣柜布局（3000×5200 窄宽户型）

常规「满墙衣柜 600 + 1.8m 床 ~1850 + 双床头柜 2×400 = 3250mm」在 3000mm 开间放不下。改用 L 形衣柜：

L 形短边沿北墙（主卫外墙下方），宽度约 1500-2000mm，深度 600mm
L 形长边沿东墙，利用 5200mm 进深延伸，深度 600mm
床1.8m 床靠西南角（床头朝东或朝北），不做床头柜
过道床边留 ≥600mm，L 形衣柜短边恰好遮挡主卫门
收纳量不输满墙衣柜，且解决了 3m 开间过窄的问题。需提供尺寸草图给工长，水电定位前确认。

施工总流程

1. 拆改砌墙（3-5天）
承重墙绝对不能动！新旧墙交接必须植筋+挂网。门洞必须加过梁。
2. 水电改造（5-7天）★最重要
全包最容易减配的环节。确认电线水管品牌。要求水管走顶。验收时亲自在场：打压 0.8MPa 保 30 分钟。拍照+录像存档。
3. 地暖打压
装修期间保持地暖压力表可见。如需重铺，豆石混凝土回填养护 7 天。
4. 防水工程（2-3天+闭水48h）
淋浴区墙面防水到 1.8m。卫生间门口外侧外延 50cm 做挡水坎。闭水 48h 亲自去楼下检查。
5. 瓦工贴砖（7-10天）
进场核对瓷砖品牌型号色号。要求墙地对缝。淋浴区坡度 2-3%。验收逐块敲空鼓。
6. 木工吊顶（5-7天）
轻钢龙骨不偷木龙骨。窗帘盒深度≥15cm（电动≥20cm）。石膏板拐角 L 型整板。
7. 油工（7-10天）
确认腻子和乳胶漆品牌。必须刷底漆（容易偷工）。强光手电侧面照验收无波浪。
8. 安装季（5-7天）
吊顶→橱柜→烟灶→室内门→定制柜→开关灯具→卫浴→地板→美缝→家电。严格按序。

⚡ 水电进场前必须敲定的家电

净水四级体系

级别设备过滤精度作用本方案

前置过滤器

安装位置优先装水井房（不占室内空间，反冲洗废水直接排楼道地漏）；水井房不行就装室内水表后。收房时问物业是否允许。

材质316 不锈钢滤网 + 无铅铜主体

安装位置厨房水槽下橱柜，机子尺寸约 40×20×40cm
