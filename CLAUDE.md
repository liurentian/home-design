# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概况

新房装修全流程规划项目。坐标郑州，116 平米三室两厅两卫，户型图见 `A1户型尺寸图.pdf`（AutoCAD 导出）。当前处于收房准备阶段。

## 核心约束

- **施工模式**：全包，由郑州本地小型装修工作室（家里亲人）承接。主材+辅材+人工全包，不需要自己采购瓷砖/地板/门/卫浴/橱柜等主材，但需明确包含/不包含项清单，关键材料确认品牌型号。
- **总预算**：20 万元（含硬装全包 + 家具 + 家电 + 软装 + 备用金）。
- **家庭背景**：两口之家，均不到 30 岁，无孩。偶尔父母过来小住几天。不需要儿童房设计，第三室定位为灵活多功能房（书房兼客卧）。

## 预算框架（20 万）

| 大项 | 预算 | 占比 |
|------|------|------|
| 全包硬装（设计+施工+主材+厨卫+柜体） | ¥115,000 | 57.5% |
| 家具 + 窗帘 + 灯具 | ¥25,000 | 12.5% |
| 家电 | ¥40,000 | 20% |
| 备用金 | ¥20,000 | 10% |

## 核心领域（按时间顺序）

1. **验房** — 空鼓、裂缝、渗漏、门窗、水电管路等检查项
2. **收房** — 文件核验、面积实测、物业交割
3. **设计** — 平面布局（年轻两口视角）、现代简约/奶油风、收纳、灯光、插座
4. **施工** — 全包模式下的监管要点、合同范围界定、各阶段验收标准
5. **家电选购** — 预算优先型全屋家电清单
6. **预算控制** — 20 万分配方案、全包合同避坑、亲人装修注意事项

## 输出目标

最终生成本地 HTML 文件 `index.html`（即装修规划手册），可直接在浏览器中查看或通过 GitHub Pages 分享。要求：
- 清晰的导航结构（按时间阶段组织）
- 现代化视觉设计（卡片式布局）
- 预算追踪仪表盘
- 可勾选 checklist
- 施工时间线
- 打印友好样式

## 工作方式

- 所有规划内容、清单、预算数据以 Markdown 存储在 `docs/` 目录
- 预算数据以 JSON 格式存储（`docs/budget.json`）便于程序化追踪
- 内容更新后同步刷新 HTML 文件
- 每次聚焦一个阶段逐步完善
